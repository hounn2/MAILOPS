# Web界面AI功能使用说明

## 🎉 新功能概览

现在你可以在Web界面中配置和使用AI智能回复功能了！

## 📱 Web界面功能

### 1. AI智能回复配置页面

在左侧菜单中新增了 **"AI智能回复"** 选项，包含：

#### LMStudio AI引擎配置
- ✅ 启用/禁用AI功能
- ✅ 配置服务器地址（默认: http://localhost:1234）
- ✅ 设置超时时间
- ✅ 自定义AI角色（系统提示词）
- ✅ 测试连接状态

#### 知识库管理
- ✅ 启用/禁用知识库
- ✅ 配置知识库路径
- ✅ 设置搜索文档数
- ✅ 文件上传功能（支持txt、md、pdf、docx）
- ✅ 文件删除功能
- ✅ 查看知识库文件列表

#### AI测试功能
- ✅ 输入测试邮件内容
- ✅ 选择是否使用知识库
- ✅ 实时查看AI生成的回复

### 2. 规则管理中支持AI动作

在规则编辑界面，动作类型新增：
- **🤖 AI智能回复**

可配置参数：
- 使用知识库（是/否）
- 包含原始邮件（是/否）
- 回复主题模板
- 温度参数（0-1，控制AI创造性）

## 🚀 快速开始

### 1. 启动Web服务

```bash
python web_app.py
```

### 2. 打开浏览器

访问: http://localhost:5000

### 3. 配置AI功能

1. 点击左侧菜单 **"AI智能回复"**
2. 启用 **"LMStudio AI引擎"**
3. 确保LMStudio已启动并运行服务器
4. 点击 **"测试连接"** 验证是否正常
5. 启用 **"知识库"**（可选）
6. 上传知识库文件或配置知识库路径
7. 点击 **"保存AI配置"** 和 **"保存知识库配置"**

### 4. 测试AI回复

1. 在 **AI测试** 区域输入测试邮件内容
2. 选择是否使用知识库
3. 点击 **"生成回复"**
4. 查看AI生成的回复效果

### 5. 创建AI回复规则

1. 点击左侧菜单 **"规则管理"**
2. 点击 **"新建规则"**
3. 设置条件（如：主题包含"咨询"）
4. 添加动作：选择 **"🤖 AI智能回复"**
5. 配置动作参数
6. 保存规则

### 6. 运行邮件处理

点击左侧 **"立即执行"** 按钮，AI将自动处理匹配的邮件。

## 📝 配置示例

### 完整的AI配置（config.json）

```json
{
  "lmstudio": {
    "enabled": true,
    "base_url": "http://localhost:1234",
    "timeout": 60,
    "system_prompt": "你是一个专业的客服助手。请根据客户邮件内容和知识库信息，生成礼貌、专业的回复。"
  },
  "knowledge_base": {
    "enabled": true,
    "path": "knowledge_base",
    "search_top_k": 3
  }
}
```

### AI回复规则示例

```json
{
  "id": "rule_ai",
  "name": "AI智能客服",
  "enabled": true,
  "conditions": {
    "match_all": true,
    "items": [
      {
        "field": "subject",
        "operator": "contains",
        "value": ["咨询", "问题"]
      }
    ]
  },
  "actions": [
    {
      "type": "ai_reply",
      "use_knowledge_base": true,
      "include_original": false,
      "subject": "RE: {original_subject}",
      "temperature": 0.7
    }
  ]
}
```

## 🔧 高级功能

### 1. 批量上传知识库文件

1. 进入 **AI智能回复** 页面
2. 点击 **"管理文件"**
3. 点击 **"上传文件"** 按钮
4. 选择多个文件（支持txt、md、pdf、docx）
5. 文件自动保存到知识库目录

### 2. 实时AI测试

在配置好AI后，可以实时测试效果：
1. 输入客户可能发送的邮件内容
2. 查看AI生成的回复
3. 不满意可以调整知识库或系统提示词
4. 再次测试直到效果满意

### 3. 多规则组合

可以创建多个AI规则，针对不同场景：

```
规则1: 技术支持类邮件 → AI使用技术文档知识库
规则2: 售前咨询类邮件 → AI使用产品手册知识库
规则3: 售后问题类邮件 → AI使用售后政策知识库
```

## 🛠️ 故障排除

### AI连接失败
**现象**: AI状态显示"未连接"
**解决**:
1. 确认LMStudio已启动
2. 确认LMStudio中已加载模型
3. 确认服务器地址和端口正确
4. 点击"测试连接"查看详细错误

### 知识库不生效
**现象**: AI回复未使用知识库内容
**解决**:
1. 确认知识库已启用
2. 确认知识库文件已正确上传
3. 检查文件格式是否支持
4. 查看日志确认文档是否被正确加载

### AI回复质量差
**优化方法**:
1. 完善知识库文档内容
2. 调整系统提示词，明确角色和要求
3. 调整温度参数（降低使回答更稳定）
4. 确保知识库包含相关主题
