# Outlook邮件助手系统介绍

## 一、系统概述

Outlook邮件助手是一款智能化的邮件自动处理系统，能够根据预设规则自动甄别邮件并执行相应操作。系统支持多平台运行，提供可视化Web管理界面，并集成AI智能回复功能，大幅提升邮件处理效率。

## 二、核心功能

### 2.1 智能规则引擎

系统内置强大的规则匹配引擎，支持多条件组合判断：

**条件字段**
- 邮件主题关键词匹配
- 发件人地址筛选
- 邮件正文内容检索
- 附件存在性检测
- 接收时间范围设定

**操作动作**
- 自动回复邮件（支持模板）
- 转发至指定人员
- 移动到目标文件夹
- 标记已读状态
- AI智能生成回复

### 2.2 AI智能回复

基于本地部署的大语言模型，实现智能化邮件处理：

**技术特点**
- 本地LMStudio部署，数据隐私安全
- 知识库文档检索，精准匹配问答
- 自定义AI角色，适配业务场景
- 多文档综合推理，生成专业回复

**工作流程**
```
接收邮件 → 知识库检索 → AI生成回复 → 自动发送
```

### 2.3 Web可视化配置

提供直观的管理界面，零代码完成所有配置：

**功能模块**
- 规则可视化编辑（增删改查）
- 知识库文件管理（上传删除）
- AI参数实时配置
- 执行日志详细记录
- 自动执行状态监控

### 2.4 跨平台支持

**Windows版本**
- 直接调用Outlook COM接口
- 响应速度快，实时处理
- 无需额外配置，开箱即用

**macOS版本**
- 基于Microsoft Graph API
- 支持Microsoft 365账户
- 云端同步，跨设备访问

### 3.2 技术栈

**后端技术**
- Python 3.8+（核心开发语言）
- Flask框架（Web服务）
- SQLite数据库（日志存储）
- Threading多线程（异步处理）

**前端技术**
- HTML5 + CSS3（界面构建）
- Bootstrap 5（UI框架）
- Vanilla JavaScript（交互逻辑）
- Bootstrap Icons（图标库）

**AI集成**
- LMStudio（本地大模型服务）
- OpenAI API兼容接口
- 相似度匹配算法（知识库检索）

### 3.3 模块说明

**核心模块**
```
outlook_assistant_win_standalone.py  Windows主程序
outlook_assistant_mac.py             macOS主程序
web_app.py                           Web服务程序
```

**功能模块**
```
rules.py         规则引擎（条件匹配、动作执行）
actions.py       邮件操作（回复、转发、移动）
ai_engine.py     AI引擎（LMStudio集成）
knowledge_base.py 知识库管理（文档加载、检索）
qa_engine.py     问答引擎（模糊匹配）
```

**配置文件**
```
config.json          主配置文件（规则、模板、设置）
config_with_ai.json  AI功能配置示例
qa_database.json     问答库配置
requirements_*.txt   依赖清单
```

## 四、使用指南

### 4.1 环境准备

**Windows环境**
```
1. 安装Python 3.8或更高版本
2. 安装Microsoft Outlook客户端
3. 确保Outlook正常运行并登录账户
```

**macOS环境**
```
1. 安装Python 3.8或更高版本
2. 准备Microsoft 365账户
3. 在Azure Portal注册应用程序
```

### 4.2 安装步骤

**步骤一：下载项目**
```bash
git clone https://github.com/your-repo/outlook-assistant.git
cd outlook-assistant
```

**步骤二：安装依赖**
```bash
Windows系统:
pip install -r requirements_windows.txt

macOS系统:
pip install -r requirements_mac.txt
```

**步骤三：配置系统**
```bash
cp config_with_ai.json config.json

编辑config.json:
- 配置邮箱账户信息
- 设置处理规则
- 配置AI参数（可选）
```

### 4.3 启动方式

**方式一：命令行运行**
```bash
Windows系统:
python outlook_assistant_win_standalone.py --once

macOS系统:
python outlook_assistant_mac.py --setup
```

**方式二：Web界面管理**
```bash
python web_app.py

在浏览器中打开:
http:/localhost:5000
```

**方式三：自动后台运行**
```bash
在Web界面中开启"自动执行"开关
系统将按照设定间隔自动处理邮件
```

### 4.4 规则配置示例

**示例一：客户咨询自动回复**
```json
{
  "id": "rule_001",
  "name": "客户咨询处理",
  "enabled": true,
  "conditions": {
    "match_all": true,
    "items": [
      {
        "field": "subject",
        "operator": "contains",
        "value": ["咨询", "问题"]
      },
      {
        "field": "sender_domain",
        "operator": "not_in",
        "value": ["@company.com"]
      }
    ]
  },
  "actions": [
    {
      "type": "ai_reply",
      "use_knowledge_base": true
    }
  ]
}
```

**示例二：紧急邮件转发**
```json
{
  "id": "rule_002",
  "name": "紧急邮件处理",
  "enabled": true,
  "conditions": {
    "match_all": true,
    "items": [
      {
        "field": "subject",
        "operator": "contains",
        "value": ["紧急", "重要"]
      }
    ]
  },
  "actions": [
    {
      "type": "forward",
      "to": ["manager@company.com"],
      "subject_prefix": "[紧急] "
    }
  ]
}
```

### 4.5 AI功能配置

**步骤一：安装LMStudio**
```
1. 从官网下载LMStudio
2. 安装并启动应用程序
3. 下载所需语言模型
4. 启动本地服务器（默认端口1234）
```

**步骤二：准备知识库**
```
1. 创建knowledge_base目录
2. 放入业务文档（支持txt、md、pdf格式）
3. 在Web界面上传文件
```

**步骤三：启用AI回复**
```
1. 在Web界面进入"AI智能回复"页面
2. 启用AI引擎并配置参数
3. 测试连接确保正常
4. 在规则中添加AI回复动作
```

### 4.6 日常维护

**查看执行日志**
```bash
Windows系统:
type outlook_assistant.log

macOS或Linux系统:
tail -f outlook_assistant.log
```

**备份配置文件**
```bash
cp config.json config.json.backup
```

**更新知识库**
```bash
1. 在Web界面上传新文档
2. 系统会自动重新加载
3. 测试AI回复效果
```

## 五、常见问题

**问题一：无法连接Outlook**
- 确保Outlook客户端已启动
- 检查账户是否正常登录
- 确认未开启只读模式

**问题二：AI回复不生效**
- 检查LMStudio是否正常运行
- 验证知识库文件是否正确加载
- 查看日志中的错误信息

**问题三：规则不触发**
- 检查规则是否已启用
- 验证条件设置是否正确
- 确认邮件符合所有条件
